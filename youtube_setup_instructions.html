Step 1: Update YouTube Video IDs

Edit the file youtube_config.php and replace the placeholder video IDs with your actual YouTube video IDs:

// In youtube_config.php, replace these: 'youtube_id' => 'YOUR_VIDEO_ID_1', // ‚Üê Replace with actual video ID 'youtube_id' => 'YOUR_VIDEO_ID_2', // ‚Üê Replace with actual video ID // etc... // Example: 'youtube_id' => 'dQw4w9WgXcQ', // Actual YouTube video ID

How to get YouTube video IDs:

  • Go to your YouTube video
  • Look at the URL: youtube.com/watch?v=dQw4w9WgXcQ
  • The part after v= is your video ID

Step 2: Get YouTube API Key (Optional but Recommended)

To automatically fetch video details and thumbnails:

  1. Go to Google Developers Console
  2. Create a new project or select existing one
  3. Enable "YouTube Data API v3"
  4. Create credentials ‚Üí API Key
  5. Copy the API key to youtube_config.php:
define('YOUTUBE_API_KEY', 'YOUR_ACTUAL_API_KEY_HERE'); define('YOUTUBE_CHANNEL_ID', 'YOUR_CHANNEL_ID_HERE');

Step 3: Test Your Setup

  1. Update the video IDs in youtube_config.php
  2. Delete current videos from database (optional - they'll be updated automatically)
  3. Visit your Live TV page to see your videos
  4. Use the admin panel to manage additional content
